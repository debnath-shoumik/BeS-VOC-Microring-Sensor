# Polarization-Selective BeS-Clad Microring Resonator for VOC Fingerprinting

This repository contains the simulation files, material datasets, and post-processing scripts associated with the research paper:

> "Single-Device VOC Fingerprinting via Polarization-Selective Anisotropic BeS-Clad Microring Resonator"

## Authors
- Sudipta Saha
- Shoumik Debnath
- Md Kawsar Alam

## Abstract
This work presents a silicon microring resonator with an anisotropic BeS sensing layer for polarization-selective volatile organic compound (VOC) detection. TE and TM modes probe orthogonal components of the BeS permittivity tensor, enabling intrinsic dual-channel optical fingerprinting from a single device.
